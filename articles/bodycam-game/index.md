# BodyCam系ゲームの原点、Unrecordを知っていますか？


注意: この記事には銃撃シーンを含むゲーム画像・動画へのリンクが含まれています。

リアルすぎるFPS

2024年、ゲーム業界に1つのトレンドが巻き起こりました。それはボディカメラ視点を採用した超リアルなFPS、通称「BodyCam系ゲーム」の流行です。

Unrecord ストアページより

ボディカメラ――警察官や軍人が胸元に装着する小型カメラの映像を模したこのジャンルは、UnrealEngine5の圧倒的なグラフィック性能と相まって、「これは本物の映像なのか、それともゲームなのか」という議論を巻き起こすほどの没入感を提供しています。

成功を収めた『Bodycam』

現在、このジャンルで最も成功しているのは、Reissad Studioが開発した『Bodycam』でしょう。2024年6月7日にSteamで早期アクセスを開始したこのタイトルは、リリース初週でSteam売上ランキング1位を獲得し、大きな成功を収めています。それに加えて、2024年末から2025年にかけて、多くのBodyCam系ゲームが登場しました。これらのタイトルは『Bodycam』の成功に便乗し、市場に次々と投入されています。

『Unrecord』を知っているか

ここで一つの事実を指摘させてください。このトレンドの火付け役は『Bodycam』ではありません。その役割を果たしたのは、DRAMA Studioが開発中の『Unrecord』です。

2023年4月、『Unrecord』の開発者Alexandre Spindler氏がYouTubeに投稿したゲームプレイトレーラーは、瞬く間に500万回近い再生を記録しました。その映像のリアルさに「これは実写をゲーム風に加工したフェイクではないか」という疑惑が巻き起こったほどです[1]。これに対し、開発者はUnrealEngine5のエディタ画面や開発中ゲームの実行ファイルを公開し、ゲームが実際にプレイ可能であることを証明しました。

私も当時公開された実行ファイルをダウンロードして動作させてみましたが、確かにそれはゲームであり、圧倒的なグラフィックと没入感の前に、視点を動かすだけで感動したことを覚えています。

Unrecordはまだ発売されていない

ここからが本題です。『Unrecord』がインターネットを席巻してから、すでに2年半以上が経過しました。しかし、このゲームは2026年1月の現在も未だリリースされていません。2025年11月、約11ヶ月ぶりに開発者のTwitterアカウントが更新され、「Still alive」という短いメッセージが投稿されました[2]。同年12月には、開発チームが10人規模に拡大し、投資家から250万ドルの資金調達に成功したこと、本格的な開発はこれからであり最終形態の披露は2026年になるとのことが語られています[3]。開発チームは、「急いでリリースして利益を得ることが目標ではない」とも明言しています[3]。

多くのゲーマーが知らないこと

現在、BodyCam系ゲームに興味を持つゲーマーで『Bodycam』の存在は知っているが、その流行の原点である『Unrecord』については知らないという方も多いのではないでしょうか。これは珍しいケースです。通常、あるジャンルを切り開いた「原点」はたとえ後発に市場を取られたとしても、その名は広く知られ、称えられるものです。しかし『Unrecord』の場合、トレーラー公開から実際のリリースまでの空白期間があまりにも長いため、当時の文脈を知らないゲーマーが増えつつあります。

さらに問題なのは、『Unrecord』が『Bodycam』と混同されてしまっていることです。見た目が非常に似ているため、『Unrecord』の存在を知っている人でさえ、『Bodycam』と誤解しているケースが少なくありません。それだけではありません。2024年末から2025年にかけて、『Bodycam』の成功に便乗した粗製濫造のBodyCam系ゲームが市場に溢れました。ストアページを見れば、似たようなサムネイル、似たような名前のタイトルが並んでいます。その中に埋もれてしまい、『Unrecord』もまた「その手の便乗作品の一つ」と見なされている可能性があります。

上記は『Bodycam』リリース日にUnrecord公式が行ったツイートです。明らかに『Bodycam』と『Unrecord』の混同に対する声明と取れる内容となっています。

原点であるにもかかわらず、後発作品やそのフォロワーと同列に扱われてしまっており、『Unrecord』にとっては不本意な状況でしょう。

その時を待とう

『Unrecord』は多くの派生作品とは異なり、シングルプレイヤーのストーリー重視ゲームとして開発されています。このことから、『Unrecord』はBodyCam系ゲームの中でも独自の位置を占めることになるでしょう。もしあなたがBodyCam系ゲームに興味を持ち、楽しんでいるのであれば、ぜひ『Unrecord』のSteamストアページも覗いてみてください。このジャンルを生み出した原点が、2026年にどのような形で世に出るのか。その行方を見守る価値は十分にあると思います。
